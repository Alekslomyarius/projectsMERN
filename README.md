# projectsMERN
# Quick Start
### Install server dependencies
```
npm install
```
### Install client dependencies
```
cd client
npm install
```
### Run both Express & React from root
```
npm run dev
```
##### Linux/Unix
```
NODE_ENV=production node server.js
```
##### Windows Cmd Prompt or Powershell
```
$env:NODE_ENV="production"
node server.js
```
